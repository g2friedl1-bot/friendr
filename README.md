# Friendr

A friend-making app that allows people to connect with those around them and creates a safe space for everyone.

## How to use

1. Use the chatbot to allow you to talk to someone.
2. Identifies triggers and sends you to BetterHelp.
3. Lets you talk to other people.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment (Vercel)

This app is ready to deploy to Vercel. Connect your repo at [vercel.com](https://vercel.com) and it will auto-detect Next.js.
